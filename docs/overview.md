# BarterShops Overview

## Introduction
BarterShops is a Bukkit plugin that facilitates player-to-player item trading using chests and signs. By leveraging intuitive sign interfaces and integrated storage management, it offers a user-friendly experience for setting up barter shops in Minecraft.

## Features
- **Chest-based Trading:** Interact with chests to initiate trades.
- **Sign Interfaces:** Configure and manage shops using in-game signs.
- **Flexible Modes:** Seamless transitions between setup, type, display, and deletion modes.
- **Permission Control:** Granular permissions to ensure secure interactions.

## Usage
1. **Creating a Shop:** Place a chest and attach a sign. Write `[barter]` on the first line of the sign.
2. **Interacting:** Shop owners can configure shops with left/right clicks while customers access shops for trading.
3. **Commands:** Use `/shop list`, `/shop nearby`, and `/shop reload` for plugin management.
